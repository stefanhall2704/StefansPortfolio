## Installation

### Install npm
[NPM Install Documentation](https://kinsta.com/blog/how-to-install-node-js/)

## Go into Directory
```bash
  cd StefansPortfolio
```
## Install Tailwind
```bash
  npm install -D tailwindcss
  npx tailwindcss init
```
